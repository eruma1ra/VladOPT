1) Общая информация

Название: ВладОПТ
Тип сайта: каталог-витрина (без онлайн-оплаты и без отображения цен)
Цель: показать ассортимент, упростить запрос на покупку/опт через контакты/Telegram/форму

2) Структура сайта
2.1. Страницы

Главная

Верх: баннер-карусель (3–5 слайдов)

Ниже: блок “Каталог” (карточки товаров/категорий, переход в каталог)

Доп. блоки по минимуму: “Как заказать” (3 шага), “Контакты” (коротко)

Каталог товаров

Список товаров + фильтры

Карточки без цен

Поиск

Пагинация/ленивая загрузка (по реализации)

Страница товара

Фото (галерея)

Название, артикул, категория

Краткие характеристики (по факту)

Наличие (опционально, если нужно) — можно “В наличии/Под заказ”

Кнопка “Купить/Запросить опт” (логика ниже)

Информация / О компании (1 страница, строго по факту)

Кто мы, что продаем, география/доставка, условия опта, реквизиты (если нужно)

Без маркетинговых текстов

Контакты (можно отдельной, можно в футере + попап)

Телефон, email, адрес (если есть)

Ссылка на Telegram

График работы (если есть)

3) Функциональные требования (Frontend)
3.1 Каталог и карточки

Карточка товара содержит:

Фото (превью)

Название

Артикул (обязательно)

Короткий параметр/категория (опционально)

Кнопка “Купить / Запросить”

Цены не показывать нигде (ни в карточках, ни в карточке товара)

3.2 Фильтры (минимально необходимые)

Категория (например: Вентили / Инструменты / Подкатегории)

Производитель/бренд

Диаметр/размер (если актуально)

Тип/назначение (если актуально)

Материал (если актуально)

Резьба/подключение (если актуально)

Наличие (если используется)

Поиск: по названию и артикулу.

Фильтры должны быть настраиваемыми через админку (какие поля участвуют).

3.3 Логика кнопки “Купить”

При нажатии открывается модальное окно:

Вариант А (приоритетный):
Показать контакты + кнопка “Написать в Telegram” (с deep-link)

Вариант Б (дополнительно):
Форма заявки: Имя, Телефон, Компания (опционально), Комментарий, поле “Какой товар” (автозаполняется выбранным товаром), согласие с обработкой данных.

После отправки формы:

Сообщение “Заявка отправлена”

Данные уходят в админку + на email (и/или в Telegram-бот/чат — если подключаем)

4) Админ-панель (Backoffice)
4.1 Роли

Администратор (полный доступ)

(опционально) Менеджер (редактирование товаров без системных настроек)

4.2 Управление товарами

CRUD: создать/редактировать/удалить товар

Массовые действия: удалить/скрыть/изменить категорию/бренд

Импорт товаров из CSV

Экспорт товаров в CSV

Ручной ввод/редактирование (форма)

4.3 Фото и файлы

Загрузка фото товара (несколько фото)

Автосжатие/превью (желательно)

Хранение: сервер/облако (по реализации)

Привязка фото к товару при импорте CSV:

либо по ссылкам на изображения в CSV

либо по совпадению имени файла/артикула (если фото загружаются архивом — опционально)

4.4 Категории и свойства

Управление деревом категорий (родитель/дочерние)

Справочники: бренды, свойства/характеристики (атрибуты)

Возможность указать, какие атрибуты участвуют в фильтрах

4.5 Заявки

Список заявок (дата, товар, имя, телефон, статус)

Статусы: Новая / В работе / Закрыта

Экспорт заявок в CSV (желательно)

5) CSV: формат (минимальный)

Обязательные поля:

sku (артикул, уникальный)

name (название)

category (путь или название)

brand (если есть)

description_short (кратко)

images (список URL через запятую / или одно URL)

Опционально:

attributes (например JSON или отдельными колонками: diameter, material, type и т.п.)

availability

Правила:

Если sku существует — обновлять товар

Если нет — создавать

Ошибки импорта логировать (строка, причина)

6) Дизайн и UI требования

Стиль: современный, спокойный, “чистый”

Цвета: синие спокойные (основной), белый/серый фон, акцентные кнопки

Адаптивность: мобильные/планшеты/десктоп

Главная: карусель-баннер сверху, ниже — каталог (видимые карточки)

Шапка: логотип “ВладОПТ”, меню (Главная / Каталог / Информация / Контакты), кнопка Telegram/Позвонить

Футер: контакты, Telegram, копирайт

7) Технические требования

Скорость: оптимизация изображений, кеширование

SEO базовое:

ЧПУ-ссылки (категории/товары)

meta title/description (шаблоны + редактирование в админке опционально)

sitemap.xml, robots.txt

Безопасность:

Авторизация в админке

Защита формы (captcha/антиспам минимум)

Аналитика: подключение (Я.Метрика/GA — по желанию)

Резервное копирование БД (желательно)

8) Контент от заказчика (что нужно предоставить)

Логотип (или текстовый временно)

Контакты: телефон, email, Telegram-ссылка, адрес/график (если есть)

CSV с товарами (или структура для наполнения)

Фотографии (или ссылки на фото)

Тексты для страницы “Информация” (коротко по факту)

9) Критерии готовности

Каталог работает: поиск + фильтры + карточки без цен

“Купить” открывает контакты/Telegram и/или форму, заявки сохраняются

Админка: ручной ввод + импорт/экспорт CSV, загрузка фото, редактирование атрибутов

Адаптивная верстка, баннер-карусель на главной

Сайт стабильно работает и быстро грузится
